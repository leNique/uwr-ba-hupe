# uwr-ba-hupe
Unterwasserrugby Hupe. Basierend auf Arduino UNO (Nachbau). Mit Bluetooth Modul.

Projekt gestartet Anfang 2017. Erster Test voraussichtlich beim Bamberger Reiter 2017 (11.03.2017).
